Developer Setup

Maintains tools and standards across all developers and vendors

Responsibilities include:

  1. Owns development setup process and tools
  2. Works with tech leads to ensure developers are set up correctly on each implementation project
  3. Works with vendors to get vendor team members access to our systems and documentation
  4. Maintains records and documentation for development setup

Coding Standards

Enforce, automate, and document coding standards.

Responsibilities include:

  1. Coordinates work of architects into cohesive standards requirements
  2. Maintains and develops standards-supporting tools

Documentation

Collect and present documentation across all NBCUTech teams

Responsibilities include:

  1. Provides a unified documentation interface/service
  2. Owns documentation technology

Tracking Systems

Facilitate internal tracking of properties and employees

Responsibilities include:

  1. Maintains a complete list of all sites with detailed information and live stats on each property
  2. Creates and maintains an employee-tracking system
  3. Owns and maintains cross-team and cross-brand ticket tracking system
  4. Expand Ideas and Feedback system
